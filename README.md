# Xvideo

[中文](README.md) | [English](README.en.md)

Xvideo 是一个原生 macOS 追番和影视客户端，用 SwiftUI 写界面，通过用户配置的媒体 API 拉取影片列表、详情和播放地址。

项目现在主要围绕“找番、追番、看番”体验在迭代：浏览首页更新、使用标签找番、查看时间表、切换剧集、记录本地追番进度、管理离线缓存和播放器设置。

## 界面预览

主界面采用 Animeko 风格的追番导航：左侧提供首页、找番、时间表、继续观看、收藏、离线缓存和设置，右侧按当前入口显示更新流、标签筛选、时间表或播放器详情。

![Xvideo macOS 应用界面预览](Docs/images/app-preview-blurred.png)

## 主要功能

- Animeko 式主导航：首页、找番、时间表、继续观看、收藏、离线缓存、设置
- 首页展示继续观看、今日/本周更新、最近更新、标签和分类入口
- 找番页支持标题、演员、导演、标签、年份、地区、语言和类型筛选
- 时间表基于资源站更新时间生成，缺少放送字段时会降级展示
- 查看海报、简介、地区、年份、主演、导演和更新状态
- 不内置任何数据源，首次使用需添加自己的采集接口
- 支持多数据源和播放源切换，兼容 JSON、XML 以及扁平 XML 分类接口，并显示源健康状态
- 详情页显示播放源卡片、剧集列表、上次观看标记和下一集建议
- 播放器内切换上一集、下一集，支持可配置快退/快进和播放窗口 Esc 退出
- 播放器提供基础弹幕开关、本地弹幕文本导入和缓存当前剧集入口
- 播放剧集时会在本地记录观看进度，可从“继续观看”恢复上次剧集和播放时间
- 收藏影片会记录所属数据源，并可在“我的收藏”里点击或双击进入播放器继续观看
- 离线缓存页管理直链资源下载，支持暂停、取消、重试、删除和 Finder 显示
- 设置页可管理数据源、播放器、弹幕、缓存、外观和本地数据导入导出

## 运行

本项目使用 Swift Package Manager，最低支持 macOS 14。

```bash
swift run Xvideo
```

## 打包成 macOS App

```bash
./Scripts/build_app.sh
open .build/app/Xvideo.app
```

打包脚本会生成：

```text
.build/app/Xvideo.app
```

## 项目结构

```text
Sources/Xvideo
├── App                  # 应用入口和依赖装配
├── Presentation         # SwiftUI 界面和 ViewModel
├── Domain               # 模型、协议、播放源解析
├── Data                 # API 请求和仓储实现
├── Infrastructure       # 下载、收藏等本地能力
└── Shared               # 通用扩展
```

更细的设计说明放在 [Docs/Architecture.md](Docs/Architecture.md)。

## 说明

应用不内置任何数据源，当前数据来自用户自行配置的接口。播放是否可用会受到资源状态、网络环境和播放源限制影响。启用数据源前会先验证接口可用性，验证失败时会保留当前数据源。遇到某个播放源无法播放时，可以优先切换到另一个播放源试试。
